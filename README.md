# 🌍 Travel Companion App

## 📱 Overview
The Travel Companion App is an Android application designed to help international travellers quickly convert essential values such as currency, fuel efficiency, liquid volume, distance, and temperature.

This app provides a simple and user-friendly interface with real-time conversion results.

---

## 🚀 Features

- 💱 Currency Conversion (USD, AUD, EUR, JPY, GBP)
- ⛽ Fuel Efficiency Conversion (mpg ↔ km/L)
- 🧪 Liquid Volume Conversion (Gallon ↔ Liter)
- 📏 Distance Conversion (Nautical Mile ↔ Kilometer)
- 🌡️ Temperature Conversion (Celsius, Fahrenheit, Kelvin)
- ✅ Input validation (empty & invalid inputs)
- 🔄 Identity conversion handling
- 🚫 Negative value handling

---

## 🧮 Conversion Factors Used

### Currency (Fixed 2026 Rates)
- 1 USD = 1.55 AUD
- 1 USD = 0.92 EUR
- 1 USD = 148.50 JPY
- 1 USD = 0.78 GBP

### Fuel Efficiency
- 1 mpg = 0.425 km/L

### Liquid Volume
- 1 Gallon = 3.785 Liters

### Distance
- 1 Nautical Mile = 1.852 Kilometers

### Temperature
- Celsius → Fahrenheit: (C × 1.8) + 32  
- Fahrenheit → Celsius: (F − 32) / 1.8  
- Celsius → Kelvin: C + 273.15
